## Installing Dependencies

- Python 3.8
- Pipenv
- SQL database server


## Database Setup

1) update database section of config.py with your sql credentials


2) create DB

```
- import the database export shared(/database/dataframe.sql) into your own DB 

OR

- create an SQL db named 'dataframe'
```


# Environment setup

```
- $ pipenv install
```

```
- $ pipenv shell
```
